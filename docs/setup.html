First login‚Äã

Core should now be accessible on the specified port and navigating to http://<address>:<port> will display the login page. Enter your preferred admin username and password, and click "Sign Up", not "Log In", to create your admin user for Komodo. Any additional users to create accounts will be disabled by default, and must be enabled by an admin.

Https‚Äã

Komodo Core only supports http, so a reverse proxy like caddy should be used for https.
